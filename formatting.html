Various Formatting elements are as follows:

    
           
  • <b> </b>: This tag helps to format the text by making it a bold text.
    • 
            
  • <strong> </stong>: This element defines text with strong importance.
    • 
            
  • <i> </i>: This tag helps to format the text by making it an italic text.
    • 
            
  • <u> </u>: This tag helps to make an underlined text or sentence.
    • 
            
  • <em> </em>: This element defines emphasized text. The content inside is typically displayed in italic.
    • 
            
  • <mark> </mark>:This element hightlight the text or sentence.
    • 
            
  • <small> </small>:This element defines smaller text.
    • 
            
  • <del> </del>: This element defines text that has been deleted from a document. Browsers will usually strike a line through deleted text
    • 
            
  • <ins> </ins>: This element defines a text that has been inserted into a document. Browsers will usually underline inserted text
    • 
            
  • <sub> </sub>: This element defines subscript text. Subscript text appears half a character below the normal line like H2O.
    • 
            
  • <sup> </sup>: This element defines superscript text. Superscript text appears half a character above the normal line like 1st.
